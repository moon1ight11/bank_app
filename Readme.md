# BankApp API
Серверная часть банковского приложения на Go. 
Реализует REST API для управления пользователями, банковскими счетами и транзакциями с ролевой моделью доступа.

## Архитектура
- Go 1.25 — язык разработки
- PostgreSQL — основное хранилище данных
- Redis — кэширование
- Prometheus + Grafana — мониторинг
- Docker — контейнеризация
- GitHub Actions — CI/CD

## Основные возможности
- Регистрация, аутентификация и выход пользователей (JWT в cookies)
- Ролевая модель: Basic, Verificator, Admin
- Управление банковскими счетами (создание, просмотр, удаление)
- Транзакции:
    Переводы между счетами (Basic)
    Пополнение и списание (Verificator, Admin)
- Административные функции:
    Создание администраторов и верификаторов
    Изменение ролей пользователей
- Мониторинг через Prometheus (количество запросов, ошибки, длительность, активные горутины)

## Запуск проекта
- Настройте конфигурацию в соответствии с config.example.yaml
- Запустите инфраструктурные сервисы командой docker compose up -d
- Запустите приложение
